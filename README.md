notasAjax
-------------

       getFirstNotas
       newNota
       getNotasLike
       deleteNotaById
       deleteNotaByCodigo
       updateNotaByCodigo
       updateNotaById
       getNotaById
       getNotaByCodigo
       getNotaByCodigoLike
       getNotasByTagsOr
       getNotasByTagsAnd

       getDocumentInfo
       insertDocument
       deleteDocument
